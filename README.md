# notes-zh1-hth

## 简介

本仓库用于归档和发布一系列独立 HTML 页面。这些页面以静态文件形式存储，可通过 GitHub Pages 或其他静态托管服务直接访问。仓库内容不针对特定网站或域名，仅作为个人资料整理与展示用途。

## 目录说明

```
.
├── index.html          # 入口页面（可选）
├── pages/              # 主要页面存放目录
│   ├── example-1.html
│   └── example-2.html
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录存放所有独立 HTML 页面，按功能或主题命名。
- `assets/` 目录存放页面引用的公共资源。
- 根目录下的 `index.html` 可作为导航页或默认入口。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的静态文件，可单独访问。
- 页面内容可能涉及技术笔记、工具页面、实验性演示等，不保证持续更新。
- 页面之间的链接、样式、脚本均基于相对路径，便于本地或服务器部署。

## 维护说明

- 本仓库不定期更新，新增页面或修正已有内容。
- 不保证向后兼容性，旧页面可能因资源路径变更而失效。
- 若发现页面异常，可提交 Issue 说明情况，但无法保证及时修复。
- 欢迎 Fork 或参考，但请遵守仓库内附带的许可证（如有）。
